# VotingProtocol:
Projeto desenvolvido para a disciplina de Protocolos de Comunicação da UFPE, sendo um sistema de votação seguro em uma rede WiFi não confiável.

## Install Depedencies
```bash
$ pip install -e .
```
- Principais depedencias:
  -   cryptography v3.1.1
  -   pycryptodome v3.10.1
  -   threading
  -   dataclasses
  -   datetime
  -   json
  -   secrets
  -   setuptools
